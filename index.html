BASICS OF HTML

Webpage is constructed using HTML - Hyper Text Markup Language. Contents of webpage such as docuement, image, audio or video is accessed through HTML. Fundamental building blocks are as follow,

TAGS - Tags are used to format contents in HTML page. Format of specifying any tag is < NAME > Content < /NAME >(dont' consider white spaces, during code writing) where Name is Tag name, Content is any normal content or it could other tag as well. First Name is called opening tag and other one is closing tag.
ELEMENTS- It is composed of opening tag, content & closing tag. There are two types of it Inline & Block.

Lets discuss some of tags -
1] To make any content bold, we use b tag. For Example UDACITY NANODEGREE
2] To make italics, we use em tag. For Example UDACITY NANODEGREE
3] To give line break, we use br tag. This tag is called void tag because there is no closing tag means no contents.
4] To insert link for other webpage, we use anchor tag. For Example UDACITY NANODEGREE

To better understand, take an analogy of house as a webpage. In it HTML will be like walls, kitchens, it will tell where is your kitchen, living room etc. Then CSS will add style to like coloring your walls, and other items which makes home look good.

For better and quicker understanding of HTML page structure; let's consider popular website Wikipedia. When opened in web browser Google Chrome, using Developer tool will show structure of webpage. Its called DOM abbrevation for Document object model. It's also called "Tree Structure", representation in hierarchical structure of web page.

BOX!!

From above observation of Wikipedia, webpage structure made of boxes, big boxes, small boxes within it. Thats how HTML page is constructed. Advantage of doing it that way, we can formal particular section as per our need, move it to other location easily without working on each element in that box. It saves lot of time. This is done by using HTML tag DIV.
There are several advantages of these boxes. When you have design along with some text material and you have fit that into webpage; boxes helps lot. Cut down page into separate big boxes, then further cutting down those individual boxes into another small boxes as per requirement, for example if have images in it.
